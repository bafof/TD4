Commandes Git Utiles

  • git clone <URL>

    Clone un référentiel distant vers votre ordinateur.

  • git commit -m "Message de commit"

    Enregistre les modifications de votre répertoire de travail dans l'historique des commits.

  • git push <remote> <branch>

    Pousse les commits locaux vers un référentiel distant spécifique et une branche.
